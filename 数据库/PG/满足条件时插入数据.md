# 满足条件时插入数据

例如：当表中不存在某记录时，才插入这条记录。

```sql
INSERT INTO 表名(列名1, 列名2)
SELECT
    '值1', '值2'
WHERE NOT EXISTS (
    SELECT * FROM 表名 WHERE 列名1 = '值1', 列名2 = '值2'
);
```
